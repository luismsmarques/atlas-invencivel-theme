== Description ==

Atlas Invencível is a cutting-edge WordPress theme designed for professionals, entrepreneurs, and businesses looking to showcase their portfolio, services, and achievements. Built with WordPress 2025 best practices, this theme uses traditional PHP templating for optimal performance and simplicity.

**Key Features:**

* **Pure PHP Architecture**: Traditional WordPress templating for maximum performance
* **Custom Post Types**: Projects, Skills, Timeline Items, Services, and Company Logos
* **Dynamic Content Management**: Easy-to-use admin interface for managing portfolio content
* **Options Page**: Easy-to-use admin interface for theme customization
* **Responsive Design**: Mobile-first approach with perfect display on all devices
* **Accessibility Ready**: WCAG AA compliant with semantic HTML and ARIA labels
* **Translation Ready**: Full internationalization support with .pot file
* **Performance Optimized**: Clean code, optimized assets, and fast loading times

**Perfect For:**
* Professional portfolios
* Business websites
* Service providers
* Creative agencies
* Entrepreneurs and startups
* Company showcases

== Installation ==

1. Upload the theme files to the `/wp-content/themes/atlas-theme` directory, or install the theme through the WordPress themes screen directly.
2. Activate the theme through the 'Appearance' screen in WordPress.
3. Go to Appearance > Customize to configure the theme settings.
4. Create your content using the Custom Post Types (Projects, Skills, Timeline, Services, Company Logos).
5. Use the block patterns and custom blocks to build your pages.

== Frequently Asked Questions ==

= How do I add my projects? =

Go to Projects in your WordPress admin and click "Add New". Fill in the project details including title, description, featured image, project category, and external URL if applicable.

= Can I customize the colors? =

Yes! Go to Appearance > Customize > Colors to change the primary (#134686), secondary (#FEB21A), background (#FDF4E3), and text colors of your theme.

= Does this theme work with page builders? =

Yes, Atlas Invencível is compatible with most page builders while also providing its own custom blocks and patterns for the block editor. You can use both approaches simultaneously.

= Is the theme mobile responsive? =

Absolutely! The theme is built with a mobile-first approach and looks perfect on all devices including phones, tablets, and desktops.

= Can I translate this theme? =

Yes, the theme is fully translation ready with a .pot file included. You can use translation plugins or create your own translation files using the text domain 'atlas-theme'.

= How do I add my skills? =

Go to Skills in your WordPress admin and click "Add New". Add the skill name, abbreviation, icon, and background color. These will appear in the skills grid block.

= Can I customize the hero section? =

Yes! Go to Appearance > Customize > Hero Section to customize the greeting text, your name, role, featured image, and statistics.

= How do I add timeline items? =

Go to Timeline in your WordPress admin and click "Add New". Add timeline items for education or work experience with dates, descriptions, and locations.

= What Custom Post Types are included? =

The theme includes Projects, Skills, Timeline Items, Services, and Company Logos custom post types for comprehensive portfolio management.

= Is the theme accessible? =

Yes, the theme follows WCAG AA guidelines with semantic HTML, ARIA labels, keyboard navigation support, and proper color contrast.

= Can I use this theme for a business website? =

Absolutely! The theme is perfect for professional portfolios, business websites, service providers, creative agencies, and company showcases.

= How do I import sample content? =

Use the provided sample-content.xml file with the WordPress importer (Tools > Import > WordPress) to get started with demo content.

= Does the theme support Full Site Editing? =

Yes! The theme supports both classic PHP templates and Full Site Editing (FSE) with block-based templates and template parts.
