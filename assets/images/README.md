# Como Adicionar Sua Foto Real

## 📸 **Instruções para Substituir a Foto:**

### **1. Preparar a Imagem:**
- Formato recomendado: JPG ou PNG
- Resolução: mínimo 600x800 pixels
- Proporção: retrato (vertical)
- Tamanho do arquivo: máximo 2MB

### **2. Nomear o Arquivo:**
- Salve como: `luis-marques-profile.jpg` ou `luis-marques-profile.png`

### **3. Colocar na Pasta:**
- Coloque o arquivo na pasta: `/Users/LuisMarques_1/AtlasTheme/images/`

### **4. Atualizar o Código:**
Substitua esta linha no arquivo `index.html`:
```html
<img src="https://images.unsplash.com/photo-1472099645785-5658abf4ff4e?w=300&h=400&fit=crop&crop=face" alt="Luis Marques" class="profile-img">
```

Por esta linha:
```html
<img src="images/luis-marques-profile.jpg" alt="Luis Marques" class="profile-img">
```

### **5. Dicas para uma Boa Foto:**
- Foto profissional ou semi-profissional
- Fundo neutro (branco, cinza claro)
- Boa iluminação
- Sorriso natural
- Vestimenta profissional
- Olhar direto para a câmera

### **6. Testar:**
- Abra o arquivo `index.html` no navegador
- Verifique se a imagem aparece corretamente
- Ajuste o tamanho se necessário

---

**Nota:** A imagem atual é apenas um placeholder. Substitua pela sua foto real seguindo as instruções acima.
